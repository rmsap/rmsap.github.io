Description

SigLo uses a pretrained neural network from MediaPipe to identify hand movements and then uses those inputs to manipulate the keyboard. Its applications include going through presentation slides without a remote, scrolling (on a webpage, application, etc.) without a remote, skipping through a video, and, as it is updated, most computer applications that can be controlled with a keyboard. I and three others developed SigLo for HackBeanpot 2022 and won the Most Innovative/ Technically Challenging award.


Development Process

When we first were brainstorming a project to make for the hack-a-thon, machine learning and computer vision stood out as branches of computer science that we wanted to learn more about. Once we found MediaPipe's API for hand identification, the idea for SigLo quickly came to us and we started developing.
Initially, we started coding and developing an activation function for scrolling all together, but after our first test we split into a programming team and an equation optimization team. I was in the programming team and we added features besides vertical scrolling (like horizontal scrolling and simulating an arrow key press with a hand motion) within the coding window that HackBeanpot allowed.


Tech Stack

SigLo is written in Python and uses several of its APIs, like OpenCV in order to implement vision features and pynput in order to mimic keyboard inputs. It also makes use of Google's MediaPipe API, giving it access to a neural network that identifies points on the hand.


Installation

1. Clone the repository on GitHub (linked above).
2. Install dependencies: pip install mediapipe pynput
3. Run the driver file: python3 driver.py


Fun Fact
