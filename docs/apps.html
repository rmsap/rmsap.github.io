Description

The HELP App provides hotlines and resources to those struggling with drugs, alcohol, mental health, etc. in an easy to use format. Similarly, RecoverWE provides resources with an emphasis on drug/opioid misuse and recovery. Both apps are available in English and Spanish and were developed for Hunterdon Prevention Resources, a non-profit in central New Jersey.


Development Process

Development on the HELP App started in fall of 2019. Initially developed as a part of my high school's Code Club, we chose to use React Native because it provided a way to write the app in a single language and deploy it on both iOS and Android. After several design iterations and a lengthy beta test period, the HELP App was released to the public in spring 2020.
During summer 2021, the HELP App was significantly updated, with a database being added to the back-end to support the ability to add data to the app without needing to release on update for the app on the app stores.
During spring and summer 2022, the HELP App was again updated with a new feature. It was expanded with the ability to display information for additional counties, with Somerset county being added in addition to Hunterdon county at this time.
Also during spring and summer 2022, the RecoverWE app was developed. Its initial development was informed by what we learned from the HELP App, so we bypassed many of the problems that we encountered during the HELP App's initial development. It has a back-end database which stores information like the HELP App, but unlike the HELP App the database can be used to render images in the app.


Tech Stack

The front-ends of both apps are programmed in JavaScript and JSX using the React Native framework. Their back-ends employ a SQL database hosted on Amazon Web Services. Expo is used to compile, build, and run both apps.
