Description

The HELP App provides hotlines and resources to those struggling with drugs, alcohol, mental health, etc. in an easy to use format. Similarly, RecoverWE provides resources with an emphasis on drug/opioid misuse and recovery. Both apps were developed for Hunterdon Prevention Resources, a non-profit in central New Jersey.
